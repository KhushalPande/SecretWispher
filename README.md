# SecretWispher
Built a dynamic web application using Node.js, Express, Axios, and EJS that fetches real-time data from a REST API and renders it on the frontend.

# Secret App (Express + EJS)

## Features
- Fetches random secrets using API
- Displays secret + username
- Built using Express, Axios, EJS

## Tech Stack
- Node.js
- Express.js
- EJS
- Axios

## How to Run
npm install
node index.js

Open http://localhost:3000
